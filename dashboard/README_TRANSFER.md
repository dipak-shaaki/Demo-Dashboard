# 🚀 Nepal Life Insurance Dashboard - Quick Start Guide

## Option 1: Open Directly (NO INSTALLATION NEEDED!)

1. **Double-click** `demo-production.html` 
2. **OR** right-click → Open with → Your browser
3. **OR** drag the file into your browser

**That's it! Dashboard opens instantly!** ✅

---

## Option 2: Share on Network

```bash
# Open terminal in this folder
python3 -m http.server 8080

# Then open in browser:
http://localhost:8080/demo-production.html

# Share with others on same network:
http://YOUR_IP:8080/demo-production.html
```

---

## Option 3: Transfer to Another Computer

### Via USB/Email/Cloud:
1. Copy `demo-production.html` file
2. Paste on another computer
3. Double-click to open

### Via Network:
```bash
# On your computer:
scp demo-production.html username@other-computer:/path/

# On other computer:
xdg-open demo-production.html
```

**Charts not loading?**

- Try a different browser

**File won't open?**
- Make sure file extension is `.html`
- Right-click → Open with → Choose browser
